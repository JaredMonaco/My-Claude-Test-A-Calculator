# My Claude Test A-Calculator

A simple calculator that can add, subtract, multiply, and divide.

## How to use it

Double-click **index.html** to open the calculator in your web browser. That's it — no installation needed.

- Tap the number buttons to enter numbers
- Tap **+**, **−**, **×**, or **÷** to choose an operation
- Tap **=** to see the answer
- **C** clears everything; **⌫** deletes the last digit
